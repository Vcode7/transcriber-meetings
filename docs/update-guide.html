📋   Update Guide

AI Meeting Transcriber — Update Instructions

Follow these steps carefully to update your installation safely without losing any data or settings.

Estimated time: ~5 minutes Difficulty: Easy Requires: 3 downloaded files
📦 What You Need
🗂️
models.zip Updated AI model files for transcription
🔧
patch_vX.X_to_vX.X.zip Patch file with code & config updates
⚙️
updater.exe Automated update utility — run last
⚠️ Make sure you have all three files before starting. Do not skip any step — the order matters.
🚀 Update Steps
1
Close AI Meeting Transcriber Completely
Ensure no background processes are running before making any file changes
🛑 Critical: If the application is running while you replace files, the update will fail or corrupt your installation. You must stop all related processes first.

Open Task Manager by pressing Ctrl + Shift + Esc, then click the Processes tab and look for these two entries:

Task Manager — Processes
backend.exe ⚠ Must End
AI Meeting Transcriber.exe ⚠ Must End
  • A If you see backend.exe — right-click it and choose "End Task".
  • B If you see AI Meeting Transcriber.exe — right-click it and choose "End Task".
  • C If neither process appears, the app is already closed — proceed to Step 2.
  • D Wait a few seconds, then refresh Task Manager (F5) to confirm both entries are gone.
✅ Both processes should no longer appear in Task Manager before you continue.
2
Extract models.zip
Unzip the new AI model files to a temporary location
  • 1 Locate models.zip in your Downloads folder (or wherever you saved it).
  • 2 Right-click models.zip and select "Extract All…" (Windows built-in), or use 7-Zip / WinRAR.
  • 4 Click "Extract" and wait for it to complete — this may take a moment depending on the model size.
💡 After extraction you should see a models folder containing subfolders and .bin / .onnx / config files inside.
3
Replace the runtime\models Folder
Overwrite the existing models in your installation directory

Navigate to your AI Meeting Transcriber installation directory. The default path is:

Install Dir C:\Program Files\AI Meeting Transcriber\

Open the runtime subfolder. You will find a models folder inside:

Target Path C:\Program Files\AI Meeting Transcriber\runtime\models\
Old (to be replaced)
runtime\models\
→
New (from models.zip)
models\ ← paste here
  • 2 Copy the newly extracted models folder from your Desktop (or temp location) into the runtime\ folder.
  • 3 If Windows asks "Replace files in destination?" — click "Replace the files in the destination".
  • 4 Confirm the new models folder is now correctly placed at the path shown above.
🔒 If Windows blocks the copy due to permissions, open File Explorer using Run as administrator and repeat the copy operation.
4
Run updater.exe
Apply the patch file and finalize the update automatically
⚠️ Make sure the patch file (e.g. patch_vX.X_to_vX.X.zip) is in the same folder as updater.exe before launching.
  • 1 Right-click updater.exe and select "Run as administrator" to ensure it has permission to write to Program Files.
  • 2 If prompted by Windows Defender / SmartScreen — click "More info" then "Run anyway".
  • 3 The updater window will open. You will be prompted to select your installation path.
⚙️   AI Meeting Transcriber — Updater
C:\Program Files\AI Meeting Transcriber
patch_vX.X_to_vX.X.zip  ✓ detected
▶  Apply Update
  • 4 Click the Browse / folder icon next to the path field and navigate to:
    Default Path C:\Program Files\AI Meeting Transcriber
  • 5 Confirm the patch file is auto-detected. If not, browse to select it manually.
  • 6 Click "Apply Update" and wait for the process to complete. Do not close the updater window while it is working.
  • 7 Once the updater shows a success / complete message, click "Finish" or close the window.
✅ The updater will automatically apply all code changes, configuration updates, and register the new version on your system.
5
Launch & Verify
Start the application and confirm the update was applied successfully
  • ✓ Launch AI Meeting Transcriber from your desktop shortcut or Start Menu.
  • ✓ Start a quick test meeting or audio session to confirm transcription is working correctly.
🎉 You're all done! AI Meeting Transcriber has been successfully updated to the latest version.
🛠 Troubleshooting
?
Common Issues & Fixes
If something doesn't go as expected
  • ! Updater says "Access Denied" — Right-click updater.exe and select Run as administrator.
  • ! Application won't start after update — Restore the models_backup folder by renaming it back to models, then contact support.
  • ! Patch file not detected — Ensure the patch .zip is in the exact same folder as updater.exe.
  • ! models folder seems empty after extraction — Re-download models.zip; the download may have been incomplete or corrupted.
  • ! backend.exe keeps restarting in Task Manager — Open the app normally first, close it via the system tray icon, then check Task Manager again.
